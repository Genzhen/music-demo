# 云音乐案例项目

## 一、服务端代码  NeteaseCloudMusicApi

### 环境要求

需要 NodeJS 8.12+ 环境

### 安装

```shell
$ npm install
```

### 运行

```shell
$ node app.js
```

服务器启动默认端口为 3000,若不想使用 3000 端口,可使用以下命令: Mac/Linux

```shell
$ PORT=4000 node app.js
```

windows 下使用 git-bash 或者 cmder 等终端执行以下命令:

```shell
$ set PORT=4000 && node app.js
```

### 详细文档，查看 NeteaseCloudMusicApi/README.md
